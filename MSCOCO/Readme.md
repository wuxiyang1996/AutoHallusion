Both files are used for pre-processing the MSCOCO validation dataset. Files need to be place under the dataset folder in the following structure:
```
├── datasets ── MSCOCO
│                 └── MSCOCO Val Dataset
|                 └── obj_db_clean.json
|                 └── scene_db_clean.json
|                 └── corr_obj_db_clean.json
```
